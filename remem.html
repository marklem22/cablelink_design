REM vs EM Demonstration

REM Example

Root font size = 16px

2rem Title = 32px

Width: 20rem (320px)
Padding: 2rem (32px)

Everything is based on the root html font size.

EM Example

Parent font size = 20px

2em Title = 40px

Width: 20em (400px)
Padding: 2em (40px)

Everything is based on the parent's font size.

Nested EM Problem

Parent (20px)
Child (2em = 40px)
Grandchild (2em = 80px)

Notice how EM multiplies as it nests.

Nested REM Behavior

Parent
Child (2rem = 32px)
Grandchild (still 2rem = 32px)

REM always uses the root font size and does not multiply through nesting.

Real-World Example

.card {
    padding: 2rem;
    margin-bottom: 1rem;
}

.button {
    font-size: 1rem;
    padding: 0.5em 1em;
}

✔ Use REM for layout spacing.
✔ Use EM for button padding and component scaling.
